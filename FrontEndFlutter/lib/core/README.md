# Core Module

Ce module contient le code partagé et la configuration de base de l'application.

## Structure

- **config/** : Configuration de l'application (thème, API, etc.)
- **constants/** : Constantes utilisées dans toute l'application
- **errors/** : Gestion des erreurs et exceptions
- **network/** : Client HTTP et intercepteurs
- **utils/** : Fonctions utilitaires et helpers
- **widgets/** : Widgets réutilisables au niveau global
